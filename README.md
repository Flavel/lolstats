Это просто скрипт, который перенаправляет запросы из 127.0.0.1:8081 на 127.0.0.1:2999 и шлет ответы обратно
Для работы можно просто запустить скрипт, например в PyCharm.
Или можно сделать .exe, для этого в терминале PyCharm вводим pip install pyinstaller

Проверить с помощью:
  pyinstaller --version
Должно вывести что-то вроде этого:
  6.17.0

Жмем pyinstaller --onefile main.py
И через некоторое время main.exe появляется в /dist/main.exe, запускаем, радуемся

Или можно просто взять экзэшник из репозитория
